Question-1: What are the different ways to select an element in the DOM?

There are many ways to select elements in Dom.
1. By ID (getElementById)
2. By Class Name (getElementsByClassName)
3. By Tag Name (getElementsByTagName)
4. By Name Attribute (getElementsByName)
5. By CSS Selector (querySelector)
6. By CSS Selector (querySelectorAll)

Question-2: What is the difference between innerHTML, innerText, and textContent?

1. innerHTML: Gets or sets the HTML inside an element.
2. innerText: Gets or sets only visible text
3.textContent: Gets or sets all text

Question-3: What is event delegation in the DOM?

Event Delegation is a JavaScript technique where a single event listener is added to a parent element instead of multiple child elements. This works because of event bubbling

Question-4: What is event bubbling in the DOM?

Event Bubbling is the process where an event starts from the target element and then propagates (bubbles up) to its parent elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

Create Elements: document.createElement("tag")
Add Elements : appendChild(), prepend(), insertBefore()
Remove Elements : remove(), removeChild()
Replace Elements : replaceChild()
